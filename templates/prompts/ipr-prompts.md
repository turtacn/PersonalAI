# 知识产权(IPR)提示模板集合

## 专利分析与撰写

### 专利可专利性分析

```
请对以下发明创意进行可专利性分析：

发明名称：[发明名称]

技术领域：[技术领域]

发明概述：
[详细描述发明的技术方案、解决的问题和技术效果]

已知相关技术：
[列出您已知的最接近的现有技术]

请提供：
1. 该发明的潜在可专利点分析
2. 新颖性初步评估
3. 创造性(非显而易见性)初步评估
4. 实用性评估
5. 可能的专利分类号(IPC分类)建议
6. 提高可专利性的优化建议
7. 可能面临的驳回理由及应对策略
```

### 专利权利要求书撰写

```
请根据以下发明信息，起草一组专利权利要求：

发明名称：[发明名称]

发明领域：[技术领域]

发明概述：
[发明的详细描述，包括技术问题、解决方案和技术效果]

关键技术特征：
1. [关键特征1]
2. [关键特征2]
3. [关键特征3]
...

请撰写：
1. 一个独立权利要求，覆盖发明的核心技术方案
2. 3-5个从属权利要求，进一步限定独立权利要求的技术特征
3. 可选的其他类型权利要求(方法、装置、系统等)

请确保权利要求：
- 符合专利法规定的格式和结构
- 术语使用一致且明确
- 提供适当的保护范围
- 避免不必要的限制
- 考虑可能的规避设计
```

### 专利说明书撰写指南

```
请帮我为以下发明创建一个专利说明书大纲：

发明名称：[发明名称]

技术领域：[技术领域]

发明概述：
[发明的简要描述，包括解决的问题和核心方案]

关键组成部分/步骤：
1. [组成部分/步骤1]
2. [组成部分/步骤2]
...

请提供：
1. 说明书背景技术部分的建议内容
2. 发明内容部分的结构和要点
3. 附图说明部分的建议(需要哪些图)
4. 具体实施方式部分的组织结构
5. 应重点描述的技术细节
6. 应避免的常见说明书撰写错误
7. 确保充分公开的建议

说明书风格要求：
- 术语使用一致
- 描述清晰具体
- 支持权利要求的全部特征
- 提供多种实施方式
```

### 商标可注册性分析

```

[国际分类号及具体商品/服务描述]

目标注册国家/地区：[国家/地区]

请提供：
1. 固有显著性分析(描述性、通用性、暗示性等)
```
